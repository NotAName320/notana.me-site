About this site

I registered the domain notana.me on November 7th, 2021. The site runs on a Raspberry Pi 4 through an Apache2 web server which forwards to a Flask program through mod_wsgi. It may sound weird that I chose to host a purely static site using Flask, but stay tuned for some stuff that actually takes advantage of it!
I am not a good HTML/CSS programmer. If you have any comments or suggestions about this website, feel free to email me or shoot me a Discord message.
Note: This site uses target="_blank" on links without the rel="noopener" property. I didn't use them because I was lazy. Every modern web browser (except for Opera for some reason) will implicitly add rel="noopener" anyways. If you are viewing this site in some sort of dinosaur browser last updated before 2021 (or Opera), I'd beware of opening any links directly.
